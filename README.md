# AI-resume-Assistant
This is an Resume Helper, where this AI will tell us what changes we need to make in order to alter the resume.
AI Resume Polish Assistant 🚀
An AI-powered tool that helps job seekers tailor and refine their resumes for specific job positions. This project uses Large Language Models (LLMs) to provide professional feedback and rewritten content instantly.

🌟 Features
Context-Aware Polishing: Analyzes your resume based on a specific target job title.

Custom Instructions: Allows users to provide specific feedback (e.g., "make it sound more technical" or "fix my grammar").

Interactive UI: Built with Gradio for a clean, user-friendly web interface within the notebook.

State-of-the-Art LLM: Utilizes the Zephyr-7B-Beta model via the Hugging Face Transformers library.

🛠️ Tech Stack
Language: Python

AI Framework: Hugging Face (Transformers, Accelerate)

Model: Zephyr-7B-Beta

UI: Gradio

Environment: Google Colab (GPU accelerated)

🚀 How to Use
Open the notebook in Google Colab.

Ensure your Runtime type is set to GPU (T4 or higher).

Run all cells to install dependencies and load the model.

Enter the Position Name, paste your Resume Content, and add any specific instructions.

Click Submit to receive your polished, professional resume.
